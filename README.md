# CannyValley

An extensible Artificial Intelligence framework that utilizes Convolutional Neural Networks (CNNs) to detect whether an input image was artificially generated by a Large Language Model (LLM) or is authentic.

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Python](https://img.shields.io/badge/python-3.8+-blue.svg)](https://www.python.org/downloads/)

## 🎯 Overview

CannyValley addresses the growing challenge of distinguishing AI-generated images from authentic photographs. As generative AI models become increasingly sophisticated, the ability to detect synthetic media becomes crucial for maintaining trust, authenticity, and security in digital content.

This framework leverages deep learning techniques, specifically Convolutional Neural Networks, to analyze image patterns, artifacts, and characteristics that differentiate AI-generated content from real-world imagery.

## ✨ Features

- **🔍 High-Accuracy Detection**: State-of-the-art CNN architecture optimized for identifying AI-generated images
- **🔧 Extensible Framework**: Modular design allowing easy integration of new models and detection techniques
- **⚡ Fast Inference**: Optimized for real-time image analysis
- **📊 Detailed Analysis**: Provides confidence scores and visualization of decision factors
- **🎨 Multi-Model Support**: Compatible with images from various generative AI systems (DALL-E, Midjourney, Stable Diffusion, etc.)
- **🔌 API-Ready**: Built-in REST API for seamless integration into existing workflows
- **📈 Continuous Learning**: Support for model fine-tuning with custom datasets

## 🚀 Getting Started

### Prerequisites

- Python 3.11 or higher
- pip package manager
- (Optional) CUDA-compatible GPU for accelerated training and inference

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/CannyValley.git
cd CannyValley

# Create a virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```

### Quick Start

```python
from cannyvalley import ImageDetector

# Initialize the detector
detector = ImageDetector()

# Analyze an image
result = detector.predict("path/to/image.jpg")

print(f"AI-Generated: {result.is_ai_generated}")
print(f"Confidence: {result.confidence:.2%}")
```

## 📖 Usage

### Command Line Interface

```bash
# Analyze a single image
python -m cannyvalley detect --image path/to/image.jpg

# Batch processing
python -m cannyvalley detect --directory path/to/images/ --output results.csv

# Start the API server
python -m cannyvalley serve --port 8000
```

### Python API

```python
from cannyvalley import ImageDetector, ModelConfig

# Custom configuration
config = ModelConfig(
    model_name="cannyvalley-v2",
    threshold=0.85,
    device="cuda"  # or "cpu"
)

detector = ImageDetector(config)

# Get detailed analysis
analysis = detector.analyze("image.jpg", detailed=True)
print(analysis.feature_map)  # Visualization of detected patterns
print(analysis.artifacts)     # Identified AI artifacts
```

### REST API

```bash
# Start the server
python -m cannyvalley serve

# Make a request
curl -X POST http://localhost:8000/api/detect \
  -F "image=@path/to/image.jpg"
```

## 🏗️ Architecture

CannyValley is built on a modular architecture:

```
┌─────────────────────────────────────────┐
│          Input Layer                    │
│  (Image Preprocessing & Augmentation)   │
└──────────────┬──────────────────────────┘
               │
┌──────────────▼──────────────────────────┐
│       Feature Extraction                │
│    (CNN Backbone - ResNet/EfficientNet) │
└──────────────┬──────────────────────────┘
               │
┌──────────────▼──────────────────────────┐
│      Artifact Detection Module          │
│   (Specialized layers for AI patterns)  │
└──────────────┬──────────────────────────┘
               │
┌──────────────▼──────────────────────────┐
│      Classification Head                │
│    (Binary: Real vs AI-Generated)       │
└──────────────┬──────────────────────────┘
               │
┌──────────────▼──────────────────────────┐
│         Output Layer                    │
│  (Confidence Score + Visualization)     │
└─────────────────────────────────────────┘
```

### Key Components

- **Preprocessing Pipeline**: Handles image normalization, resizing, and augmentation
- **CNN Backbone**: Deep neural network for feature extraction
- **Artifact Detector**: Specialized module for identifying AI-specific patterns
- **Classifier**: Final decision layer with confidence scoring
- **Explainability Module**: Generates visual explanations for predictions

## 🔬 Model Training

### Training Your Own Model

```python
from cannyvalley.training import Trainer
from cannyvalley.data import ImageDataset

# Prepare your dataset
train_dataset = ImageDataset(
    real_images_path="data/real/",
    ai_images_path="data/ai_generated/",
    split="train"
)

# Configure and train
trainer = Trainer(
    model_name="cannyvalley-custom",
    epochs=50,
    batch_size=32,
    learning_rate=0.001
)

trainer.train(train_dataset)
trainer.save("models/my_model.pth")
```

### Fine-tuning

```python
# Load pre-trained model and fine-tune
detector = ImageDetector.from_pretrained("cannyvalley-v2")
trainer = Trainer.from_detector(detector)
trainer.fine_tune(custom_dataset, epochs=10)
```
